Напишите функцию pow(x,n), которая возвращает x в степени n. Иначе говоря, умножает x на себя n раз и возвращает результат.

		Пример вызовов:
		pow(3, 2) = 3 * 3 = 9
		pow(3, 3) = 3 * 3 * 3 = 27
		========================
		function pow(a, e) {
			for (var multiple = 1, i = 0; i < e; i++) {
				multiple *= a;
			}
			return multiple;
		}
