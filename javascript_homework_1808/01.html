Чему равно свойство после delete?
Какие значения будут выводиться в коде ниже?

			var animal = {
					jumps: null
				};
			var rabbit = {
					jumps: true
				};
			rabbit.__proto__ = animal;
			alert( rabbit.jumps ); // ? (1)

			delete rabbit.jumps;
			alert( rabbit.jumps ); // ? (2)

			delete animal.jumps;
			alert( rabbit.jumps ); // ? (3)
			
			Итого три вопроса.

Ответ: 1. true - просто взяло значение свойство из объекта rabbit;
2. null - значение взято из объекта animal, который указан через спец.свойство proto;
3. undefined - неопределено, т.к. это свойство удалено у всех объектов в цепочке.
